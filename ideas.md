## Ιδέα 1: "Botanical Gilded Scroll" — Art Nouveau Revival

**Design Movement:** Art Nouveau Revival με botanical στοιχεία, εμπνευσμένο από χρυσά χειρόγραφα και φυτικά μοτίβα.

**Core Principles:**
1. Οργανικές καμπύλες και φυτικά μοτίβα σε κάθε section transition
2. Χρυσός ως κυρίαρχο χρώμα accent, πάντα σε warm undertone
3. Κάθετη αφηγηματική ροή — σαν ξετύλιγμα ενός scroll/χειρογράφου
4. Ασυμμετρία στη σύνθεση, με φύλλα που "εισβάλλουν" από τα πλαϊνά

**Color Philosophy:**
- Βάση: Warm ivory (#FAF7F0) — ζεστασιά, φιλοξενία
- Κείμενο: Deep warm brown (#5C4A32) — σοβαρότητα χωρίς σκληρότητα
- Accent: Antique gold (#C9A96E) — πολυτέλεια, γιορτή
- Soft accent: Muted sage (#D4C5A9) — φυσικότητα, ηρεμία
- Highlight: Champagne (#E8D5B7) — λάμψη

**Layout Paradigm:** Single-page vertical scroll με full-width sections που εναλλάσσονται μεταξύ κεντρικού περιεχομένου και asymmetric botanical frames. Κάθε section χωρίζεται με animated SVG leaf dividers.

**Signature Elements:**
1. Animated gold leaf branches που μεγαλώνουν στα πλαϊνά κατά το scroll
2. Χρυσές λεπτές γραμμές (rules) ως section dividers, ίδιες με την πρόσκληση
3. Floating gold dots/particles στο hero section

**Interaction Philosophy:** Gentle reveal — τα στοιχεία εμφανίζονται σαν να ξεδιπλώνονται μαλακά, χωρίς abrupt transitions. Hover effects σε χρυσά στοιχεία δημιουργούν subtle shimmer.

**Animation:**
- Scroll-triggered fade-in-up για κάθε section
- Parallax effect στα botanical frames
- Countdown numbers με smooth flip animation
- Gold shimmer effect στα decorative elements

**Typography System:**
- Display: Playfair Display (italic) — για ονόματα, τίτλους
- Body: Cormorant Garamond — κομψό serif για κείμενο
- Accent: Great Vibes — καλλιγραφικό για "& ο μικρός μας"

## Ιδέα 2: "Mediterranean Warmth" — Organic Minimalism

**Design Movement:** Mediterranean Organic Minimalism — εμπνευσμένο από ελληνικό φως, ζεστά υλικά, και φυσική κομψότητα.

**Core Principles:**
1. Αφαιρετικότητα με ζεστασιά — λίγα στοιχεία, μεγάλη επίδραση
2. Φυσικά textures (linen, stone, paper grain) ως backgrounds
3. Μεγάλα κενά (whitespace) που αναπνέουν
4. Χρώμα μόνο εκεί που μετράει — χρυσός μόνο σε key moments

**Color Philosophy:**
- Βάση: Warm linen (#F5F0E8) — σαν φυσικό λινό ύφασμα
- Κείμενο: Charcoal warm (#3D3428) — μαλακό αλλά readable
- Accent: Burnished gold (#B8963E) — πιο muted, πιο organic χρυσός
- Secondary: Terracotta mist (#D4B896) — μεσογειακή ζεστασιά
- Dividers: Pale gold (#DDD0B6)

**Layout Paradigm:** Stacked cards layout — κάθε section είναι ένα "κάρτα" με ελαφρύ shadow και rounded corners, τοποθετημένη σε off-white background. Asymmetric margins — content shifts ελαφρώς αριστερά ή δεξιά ανά section.

**Signature Elements:**
1. Linen paper texture overlay σε backgrounds
2. Thin gold line borders γύρω από sections
3. Olive branch / leaf watermark motifs σε low opacity

**Interaction Philosophy:** Understated elegance — minimal animations, μόνο opacity transitions. Τα κουμπιά αλλάζουν σε χρυσό background on hover. Focus σε readability και clarity.

**Animation:**
- Subtle fade-in on scroll (opacity only, no movement)
- Smooth number counting για countdown
- Gentle scale on hover για interactive elements
- No parallax — clean and fast

**Typography System:**
- Display: Cinzel — Roman-inspired, μεσογειακό
- Body: Lora — warm serif, εύκολο στην ανάγνωση
- Accent: Tangerine — ελαφρύ script

## Ιδέα 3: "Golden Veil" — Romantic Layered Elegance

**Design Movement:** Romantic Layered Elegance — πολλαπλά layers, transparency, και depth δημιουργούν αίσθηση πολυτέλειας.

**Core Principles:**
1. Layered depth — overlapping elements, semi-transparent panels, backdrop blur
2. Χρυσός gradient ως signature visual element
3. Cinematic feel — μεγάλες εικόνες, dramatic spacing
4. Κάθε section transition νιώθει σαν αλλαγή σκηνής

**Color Philosophy:**
- Βάση: Soft cream (#FBF8F1) — ζεστό, inviting
- Deep layer: Warm beige (#EDE4D3) — depth
- Κείμενο: Dark walnut (#4A3728) — rich, warm
- Primary accent: Gold gradient (linear #C9A96E → #E8D5B7) — λάμψη
- Glass effect: rgba(255,250,240,0.7) + backdrop-blur — μοντέρνο

**Layout Paradigm:** Full-viewport sections με layered composition. Hero section παίρνει 100vh. Κάθε section έχει ένα decorative background layer (botanical pattern σε low opacity) και ένα content layer (glass-morphism card). Asymmetric — content panels offset αριστερά/δεξιά.

**Signature Elements:**
1. Glass-morphism panels με χρυσό border
2. Gold gradient text για τα ονόματα του ζευγαριού
3. Animated botanical silhouettes στα backgrounds

**Interaction Philosophy:** Cinematic reveals — sections slide in με staggered children animations. Buttons έχουν gold shimmer on hover. Maps sections έχουν elegant frame treatment.

**Animation:**
- Staggered fade-in-up για children elements μέσα σε κάθε section
- Smooth parallax στα background botanical layers
- Gold shimmer/glow pulse στα CTA buttons
- Countdown με elegant flip-card style
- Intersection Observer για scroll-triggered animations

**Typography System:**
- Display: Playfair Display — bold, romantic
- Body: Source Serif 4 — clean serif, modern warmth
- Script: Dancing Script — για decorative text elements
